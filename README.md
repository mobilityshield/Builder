# MobilityShield Builder

MobilityShield Builder is the parametric design tool behind the MobilityShield project.

It is used to generate modular printable parts from editable parameters, helping turn configurable geometry into real 3D output.

## What it does

- lets users adjust part parameters
- previews the resulting geometry
- supports STL export for printing
- acts as one of the main technical proof surfaces of the project

## Project context

This repository is part of MobilityShield, an early-stage public project built around modular printable parts.

The Builder should be understood as:
- real
- live
- early-stage
- part of a broader project

It should not be read as a finished commercial platform or a complete ecosystem on its own.

## Technology

This Builder is based on **OpenJSCAD 1.10.0**.

For the underlying engine and original technical foundations, refer to the original OpenJSCAD project.
